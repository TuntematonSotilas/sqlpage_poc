## Installation 

[Download the latest SQLPage](https://github.com/lovasoa/SQLpage/releases)

And place `sqlpage.bin`  Or `sqlpage.exe` at the root of the folder

## Run

Launch in a terminal : 

`sqlpage.bin`  Or `sqlpage.exe`
