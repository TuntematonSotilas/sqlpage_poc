# sqlpage_poc
POC of SQLPage

## Documentation

https://sql.ophir.dev

## Installation 

[Download the latest SQLPage](https://sql.ophir.dev/your-first-sql-website/)

And place `sqlpage.bin`  Or `sqlpage.exe` at the root of the folder

## Run

Launch in a terminal : 

`sqlpage.bin`  Or `sqlpage.exe`
